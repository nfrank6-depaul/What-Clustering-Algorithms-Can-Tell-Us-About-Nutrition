# What-Clustering-Algorithms-Can-Tell-Us-About-Nutrition

## Description
- Collaborated with a team of 4 to explore unsupervised clustering methods and their impact on random forest and decision tree classifier performance when clustering labels were incorporated
- Methods: K-Means, Agglomerative, and DBSCAN Clustering | Random Forest and Decision Tree Classifiers
- Tools: Scikit-Learn, Pandas, NumPy, SciPy
